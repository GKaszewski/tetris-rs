# tetris-rs
My implementation of Tetris in Rust using Raylib.

## How to play
- `←` and `→` to move left and right
- `↑` to rotate
- `↓` to drop
- `space` to hard drop
- press `p` to pause

## How to build
- Install Rust
- `cargo build --release`
- `./target/release/tetris-rs`
- Enjoy!
